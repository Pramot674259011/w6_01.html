Workshop: Various Links

เรียนรู้การใช้งาน Link รูปแบบต่างๆ

1. External Links

Visit Wikipedia

target="_blank" เปิดลิ้งในแท็บใหม่

Visit Google

target="_self" เปิดลิ้งในหน้าเดิม

2. Internal Links & Anchor Links

<a href="about.html">About Us</a>
<a href="#jump">Jump to Section</a>
<h3 id="jump">This is the jump target</h3>
Click to jump

This is the jump target (ID: jump)

3. Email Link

<a href="mailto:student@example.com?subject=Hello">
   Send an Email
</a>
Send an Email

4. Download Links

<a href="file.zip" download>Download File</a>
Download Text File

*ต้องมีไฟล์ในโฟลเดอร์จริงจึงจะดาวน์โหลดได้

5. Button Links

สร้างปุ่มด้วยแท็ก button หรือใช้ CSS แต่ง Link

<button onclick="location.href='index.html'">
  Visit Website
</button>

6. Image Link

คลิกที่รูปภาพเพื่อเปิดลิงก์

<a href="https://youtube.com">
  <img src="path/to/img.jpg" width="250">
</a>

หากต้องการใช้รูปตนเอง ให้เก็บในโฟลเดอร์ เช่น images/photo.jpg
